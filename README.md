# 🎮 MATH MASTER LIFE

Math Mater Life is a simple terminal-based game built with Python where you play as a rising mathematician trying to earn money through work and mini-games.

## 🚀 Features

* 🧮 **Math Work System**
  Solve random math problems to earn money.

* 🎰 **Slot Machine**
  Try your luck and win big rewards!

* 🪙 **Coin Flip Betting**
  Gamble your money with a 50/50 chance.

* 🔁 **Game Loop System**
  Continuous gameplay with multiple choices.

---

## 🛠️ Built With

* Python 3
* Built-in libraries:

  * `random`
  * `time`

---

## 🎯 How to Play

1. Run the Python file:

   ```bash
   python mmlife.py
   ```

2. Choose an action:

   * **Work** → Solve math problems to earn money
   * **Spin the Wheel** → Play mini-games
   * **Exit** → Quit the game

3. Manage your money wisely and try to get rich!

---

## ⚠️ Notes

* The game runs in the terminal (CLI).
* No data is saved — money resets when the game restarts.
